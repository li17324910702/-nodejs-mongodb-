#### 组织结构

```
answer
├──  public       -- 公用资源
├──  views        --  art-template渲染界面文件目录 
├── models        --  mongodb数据库接口
├── app.js        --  app入口
├──  package-lock -- 依赖
└── router.js     --  路由接口
```

#### 运行

- 跳转到目录下 shell/cmd指令

```shell
node app.js
```



- 使用之前请安装所有依赖

```shell
npm install
```
